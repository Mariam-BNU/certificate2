# Ebtekar Certificate – Eye Empower Project

I am proud to have received the Ebtekar Certificate from my university for presenting an innovative project called Eye Empower.
The project was recognized for its creativity, technical implementation, and potential social impact.

# About the Project

Eye Empower is an AI-powered project designed to support and empower visually impaired individuals using intelligent computer vision and deep learning technologies.

The system focuses on improving accessibility and enhancing user independence through smart and innovative solutions.
